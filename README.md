# Arup Ratan Dey Portfolio

A modern, visually appealing portfolio website built with **React** and **Tailwind CSS** to showcase my competitive programming achievements, projects, skills, and education.

## 🖥️ Features

- **Hero Section**
  - Animated name gradient
  - Short bio and university
  - Download CV button with animation
  - Floating skill badges (⚡ CP & 💻 Dev)

- **Competitive Programming Stats**
  - Total problems solved and contests participated
  - Platform-wise stats (Codeforces, CodeChef, AtCoder, LeetCode)
  - Best ranks with links

- **Onsite Contest Achievements**
  - Animated contest cards
  - Team name, rank, year, and external standings link
  - Hover gradient and bottom border animations

- **Projects Section**
  - Project cards with image thumbnails
  - Tech stack tags
  - Live demo and GitHub links
  - Smooth hover animations

- **Skills Section**
  - Programming languages, frameworks, and tools
  - Icon-based skill cards
  - Hover effects and border animations

- **Education Section**
  - Degree, institution, year, grade
  - Certificate download/view button
  - Responsive layout for desktop and mobile

- **Footer**
  - Contact links: GitHub, LinkedIn, Email
  - Email copy-to-clipboard with feedback
  - Clean design with hover effects

---

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS
- **Icons & Animations:** Lucide-React
- **Hosting (optional):** Vercel / Netlify
- **Tools:** VS Code, Figma for UI design

## ⚡ Installation

1. **Clone the repository**

```bash
git clone https://github.com/arup-ratan-dey/portfolio.git
cd portfolio
````

2. **Install dependencies**

```bash
npm install
```

3. **Start development server**

```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 📄 Usage

* Update `src/data/profile.js` with your own details:

  * Name, bio, university
  * Competitive programming stats
  * Projects, skills, and education
  * Profile image and CV

* Update `assets/` folder with your own images or icons.

---

## 🌐 Deployment

* Build the project for production:

```bash
npm run build
```

* Deploy to **Vercel**, **Netlify**, or any static hosting service.
